Waarom heeft de volgende CSS regel: .onbekend {background-color:red;} geen effect als je deze toepast op een <td> cel in een <tr> rij met class="alt", maar wel in een <tr> rij zonder deze 'alt' class? Hint: het heeft te maken met de specificiteit van de regels

De select is gewoon veel specifieker dan die van puur de klasse .onbekend. Hoe specifieker de selectie, hoe meer invloed die krijgt.

De klasse alt heeft de volgende specificatie table tbody .alt td { background: #38F47D; color: #00496B; }
